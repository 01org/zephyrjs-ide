About

<%= APP_TITLE_HTML %> is an online JavaScript editor that lets you upload code to a board powered by Zephyr OS, directly from the browser, taking advantage of WebUSB.

Features

  • WebUSB connectivity to JavaScript Runtime for Zephyr OS powered boards
  • Instant JavaScript code execution on target
  • Integrated Monaco code editor
  • Console shell
  • Multiple tabs and Web USB device support
